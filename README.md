- Socket.io 
- Node.JS
- Express.JS

![Chat App](./preview_images/appPreview.png "Chat App Preview")

---

## Setup

```bash
git clone git@github.com:MikRyam/chat-app.git
cd chat-app

npm ci
```

## Getting Started

```bash
cd server
npm run dev

cd ..

cd client
npm run start
```
